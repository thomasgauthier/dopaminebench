## 4.2.1 Text and StyledText

OpenTUI provides two primary ways to display text: `TextRenderable` for static text and `StyledText` for text with rich formatting.

### TextRenderable
The `TextRenderable` class is used to display static text content with support for colors, attributes, and text selection . You can instantiate a `TextRenderable` with a `content` property, which can be a plain string or a `StyledText` object .

Example of `TextRenderable` with plain text and attributes:
```typescript
import { TextRenderable, TextAttributes, t, bold, underline, fg } from "@opentui/core"

const plainText = new TextRenderable(renderer, {
  id: "plain-text",
  content: "Important Message",
  fg: "#FFFF00",
  attributes: TextAttributes.BOLD | TextAttributes.UNDERLINE, // bitwise OR to combine attributes
  position: "absolute",
  left: 5,
  top: 2,
})
``` 

### StyledText
`StyledText` allows for more complex text styling using template literals and style functions . The `StyledText` class holds an array of `TextChunk` objects, where each chunk can have its own text, foreground color, background color, and attributes .

You can create `StyledText` using the `t` template literal function, along with style functions like `bold`, `underline`, and `fg` (foreground color) .

Example of `TextRenderable` with `StyledText`:
```typescript
const styledTextRenderable = new TextRenderable(renderer, {
  id: "styled-text",
  content: t`${bold("Important Message")} ${fg("#FF0000")(underline("Important Message"))}`,
  position: "absolute",
  left: 5,
  top: 3,
})
``` 

The `StyledText` object is composed of `TextChunk`s . Each `TextChunk` can define its own `fg` (foreground color), `bg` (background color), `bold`, `italic`, `underline`, `strikethrough`, `dim`, `reverse`, and `blink` attributes .

The `t` function processes template literals and values to create `TextChunk`s . Style functions like `bold`, `italic`, `underline`, `strikethrough`, `dim`, `reverse`, and `blink` apply specific attributes to the input text . Color functions `fg` and `bg` apply foreground and background colors respectively .
